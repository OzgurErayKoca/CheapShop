# CheapShop — Örnek E-ticaret

CheapShop, bu depo içindeki örnek e-ticaret tek sayfa şablonudur.

Nasıl çalıştırılır:

- `PROJECT1/index.html` veya kök `index.html` dosyasını tarayıcıda açın. Kök `index.html` otomatik olarak `PROJECT1` içine yönlendirir.

Özellikler:

- Ürün listesi ve arama
- Basit sepet (localStorage tabanlı)
- Responsive tasarım

Değiştirmek isterseniz:

- Ürünleri `PROJECT1/index.html` içindeki `products` dizisinde güncelleyin.
- Stilleri `PROJECT1/styles.css` içinde özelleştirin.
