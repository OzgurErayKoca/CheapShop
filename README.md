# CheapShop — Örnek E-ticaret


CheapShop, bu depo içindeki örnek e-ticaret tek sayfa şablonudur. Bu sürüm yalnızca HTML ve CSS içerir — tüm JavaScript kodları kaldırıldı.

Nasıl çalıştırılır:

- `index.html` dosyasını tarayıcıda açın.

Özellikler:

- Statik ürün listesi (JS yok)
- Arama ve sepet işlevleri devre dışı (sadece görünüm)
- Responsive tasarım

Değiştirmek isterseniz:

- Ürün içeriklerini `index.html` içindeki ürün kartlarında düzenleyin.
- Stilleri `styles.css` içinde özelleştirin.
