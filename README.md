# CheapShop — Örnek E-ticaret

CheapShop, bu depo içindeki örnek e-ticaret tek sayfa şablonudur.

Nasıl çalıştırılır:

- `index.html` dosyasını tarayıcıda açın.

Özellikler:

- Ürün listesi ve arama
- Basit sepet (localStorage tabanlı)
- Responsive tasarım

Değiştirmek isterseniz:

- Ürünleri `index.html` içindeki `products` dizisinde güncelleyin.
- Stilleri `styles.css` içinde özelleştirin.
